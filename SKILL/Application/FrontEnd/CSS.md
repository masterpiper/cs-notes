# CSS
用于规定HTML的样式
样式引用规则：
- 优先级：内联>内部>外部
- 如果有多重样式的引用，则符合继承规则

## 将CSS嵌入到HTML
- 内联样式
标签属性设置
- 内部样式
使用`<style></style>`标签属性设置 
- 外部样式
使用`<link></link>`标记css文件
标签的id等于css中引用的样式名

## CSS选择器
用于选择HTML里需要，改变样式的标签

优先级：**id>class>elem**

- 元素选择器：根据html标签选取

- id选择器：为标签指定id，根据id来选取

- 类选择器：根据html标签指定的class进行选取
